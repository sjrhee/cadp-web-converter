# CADP 웹 컨버터 (CADP Web Converter)

Thales CADP SDK를 사용하여 구분자가 있는 파일(CSV, TSV, TXT)을 일괄 암호화(Protect) 및 복호화(Reveal)하는 Spring Boot 웹 애플리케이션입니다.

## 주요 기능

- **마법사형 UI**: 간편한 5단계 프로세스 (파일 업로드 \u2192 미리보기 \u2192 설정 \u2192 실행 \u2192 결과).
- **고성능 일괄 처리**: 멀티스레딩(기본 8 스레드)을 통해 대용량 파일을 효율적으로 처리합니다.
- **동적 설정**: CADP 연결 설정 및 정책을 웹 UI에서 즉시 변경할 수 있습니다.
- **대화형 미리보기**: 파일 내용을 미리 확인하고 대상 컬럼을 헤더 클릭으로 손쉽게 선택할 수 있습니다.
- **상세 통계**: 처리 시간, 소요 시간, 총 처리 라인 수 등 실행 결과를 한눈에 확인할 수 있습니다.

## 필수 요구 사항

- Java 17 이상
- Maven
- Thales CADP SDK for Java

## 설치 및 실행

1. **환경 변수 설정**: 프로젝트 루트 경로에 `.env` 파일을 생성하고 아래 내용을 작성하세요.
   ```properties
   CADP_HOST=192.168.10.100
   CADP_PORT=443
   CADP_TOKEN=your_token
   CADP_USER=your_user
   CADP_POLICY=your_policy
   ```

2. **애플리케이션 실행**:
   제공된 스크립트를 사용하여 빌드 및 실행을 진행합니다:
   ```bash
   ./build_and_run.sh
   ```
   실행 후 브라우저에서 `http://localhost:8080`으로 접속하세요.

## 사용 방법

1. **시작**: 메인 화면에서 "Get Started" 버튼을 클릭합니다.
2. **파일 업로드**: 처리할 파일을 선택하고 구분자(Delimiter)를 입력합니다.
   - *팁*: "Load Preview"를 클릭한 후 "Skip Header Row"를 체크하면 데이터 구조를 더 명확히 확인할 수 있습니다.
3. **컬럼 선택**: 암호화 또는 복호화할 대상 컬럼의 헤더를 클릭하여 선택합니다.
4. **설정 확인**: CADP 연결 정보를 확인하거나 수정합니다.
5. **프로세스 실행**: 적용할 정책(Policy)과 모드(Protect/Reveal)를 선택하고 실행합니다.
6. **결과 다운로드**: 처리 결과 통계를 확인하고 변환된 파일을 다운로드합니다.
